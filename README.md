# PokePrice Scanner Extension

This is a Chrome/Edge browser extension that allows you to identify Pokemon cards on webpages and view their market prices and trends.

## Installation
1.  Open Chrome/Edge and go to `chrome://extensions`.
2.  Enable **Developer Mode** (top right).
3.  Click **Load unpacked**.
4.  Select the `pokeprice-scanner` folder inside this directory.

## Usage
-   **Hover:** Hover over images on sites like eBay to see the "Scan" button.
-   **Right Click:** Use the context menu "Scan Pokemon Card Price".
